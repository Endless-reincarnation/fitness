# 健身计划执行助手

这是一个微信小程序 MVP 原型，目标是把健身计划变成可执行、可记录、可复盘的训练流程。

## 当前能力

- 查看官方计划库。
- 查看计划详情。
- 启用计划。
- 首页展示今日训练。
- 执行训练并记录重量、次数。
- 支持训练中快速修改休息时间，并自动启动组间休息计时。
- 保存训练历史到本地缓存。
- 记录体重并查看历史。
- 支持黑黄力量风和黑绿科技风皮肤切换。
- 支持创建本地多训练日自定义计划并启用训练。
- 完成训练后会自动轮转到下一个训练日。
- 首页支持手动切换当前训练日。
- 训练中退出后可从首页继续未完成训练。

## 目录结构

```text
docs/                 产品、数据库、页面流程文档
seed/                 计划、动作、肌群种子数据
miniprogram/          微信小程序源码
project.config.json   微信开发者工具项目配置
```

## 本地运行

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择当前目录：`E:\work\ai\fitness`。
4. AppID 可先选择测试号或使用游客模式。
5. 编译后从“计划”页启用计划，再回到“首页”开始训练。

## 当前限制

- 还没有接微信登录。
- 已接入微信云开发环境，计划库和动作库支持云端读取。
- 训练记录和体重记录支持本地保存、云端写入和失败重试。
- 计划库目前只有第一份三分化计划。
- 后台管理端还没有实现。
